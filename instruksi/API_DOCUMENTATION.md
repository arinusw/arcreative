ARTICLES API\n\n### Endpoint\n`\nGET/POST http://localhost/arcreative/api/articles.php\n`\n\n### GET - Fetch All Articles\n\n**Request:**\n`\nGET /api/articles.php\nGET /api/articles.php?category=tutorial-web\nGET /api/articles.php?status=published\nGET /api/articles.php?limit=10&offset=0\n`\n\n**Parameters:**\n| Parameter | Type | Description |\n|-----------|------|-------------|\n| `category` | string | Filter by category slug |\n| `status` | string | Filter by status (draft, published, archived) |\n| `limit` | integer | Max results per request (default: 10) |\n| `offset` | integer | Pagination offset (default: 0) |\n\n**Response:**\n`json\n{\n  \"status\": \"success\",\n  \"data\": [\n    {\n      \"id\": 1,\n      \"title\": \"Tutorial Design Grafis\",\n      \"slug\": \"tutorial-design-grafis\",\n      \"excerpt\": \"Tips dan trik design grafis...\",\n      \"category_id\": 1,\n      \"category_name\": \"Tutorial Design\",\n      \"author_id\": 1,\n      \"author_name\": \"Admin\",\n      \"featured_image\": \"img/article-1.jpg\",\n      \"read_time\": 5,\n      \"status\": \"published\",\n      \"published_at\": \"2026-02-10 10:30:00\",\n      \"views\": 245,\n      \"created_at\": \"2026-02-10 09:00:00\"\n    }\n  ],\n  \"count\": 1,\n  \"total\": 50\n}\n`\n\n### GET - Fetch Single Article\n\n**Request:**\n`\nGET /api/articles.php?id=1\nGET /api/articles.php?slug=tutorial-design-grafis\n`\n\n**Parameters:**\n| Parameter | Type | Description |\n|-----------|------|-------------|\n| `id` | integer | Article ID |\n| `slug` | string | Article slug |\n\n**Response:**\n`json\n{\n  \"status\": \"success\",\n  \"data\": {\n    \"id\": 1,\n    \"title\": \"Tutorial Design Grafis\",\n    \"slug\": \"tutorial-design-grafis\",\n    \"content\": \"<p>Konten artikel lengkap...</p>\",\n    \"excerpt\": \"Tips dan trik design grafis\",\n    \"category_id\": 1,\n    \"category_name\": \"Tutorial Design\",\n    \"featured_image\": \"img/article-1.jpg\",\n    \"read_time\": 5,\n    \"status\": \"published\",\n    \"views\": 245\n  }\n}\n`\n\n### POST - Create Article (Admin Only)\n\n**Request:**\n`\nPOST /api/articles.php\nContent-Type: application/json\nAuthorization: Bearer <token>\n\n{\n  \"title\": \"Tutorial Design Grafis\",\n  \"content\": \"<p>Konten artikel...</p>\",\n  \"excerpt\": \"Tips dan trik design\",\n  \"category_id\": 1,\n  \"featured_image\": \"img/article-1.jpg\",\n  \"status\": \"draft\"\n}\n`\n\n**Response:**\n`json\n{\n  \"status\": \"success\",\n  \"message\": \"Article created successfully\",\n  \"data\": {\n    \"id\": 2,\n    \"slug\": \"tutorial-design-grafis\"\n  }\n}\n`\n\n---\n\n## 2.
